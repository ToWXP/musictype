# musictype

A minimal typing trainer for practicing your own lyrics, styled like Monkeytype.

## How to use

1. Open `musictype.html` in your browser.
2. Add a song: title + lyrics (paste your own), artist and Spotify link are optional.
3. Click the song to start typing it.
4. See your WPM and accuracy at the end.

## Features

- Blue, minimal, dark theme
- Live WPM, accuracy, and timer
- Correct vs raw WPM (mistakes lower your score)
- Saves your songs and past results in the browser
- Search, sort, and edit your song list
- Optional Spotify player while you type

## Notes

- You add the lyrics yourself — the app doesn't fetch them automatically.
- No install needed, just open the HTML file.
